A patch enabling to read raw data in Unity. Last tested version: myo SDK v.0.9.0

Copy all the files into the "Assets/Myo/Scripts" folder of your Unity project and override the existing myo scripts there. To be sure, you might want to do a diff to the existing files to check for unwanted changes.
